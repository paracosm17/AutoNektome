# AutoNektome 🚀

![Версия](https://img.shields.io/badge/версия-5.0.0-brightgreen) ![Лицензия](https://img.shields.io/badge/лицензия-MIT-blue) ![Совместимость](https://img.shields.io/badge/сайт-nekto.me/audiochat-orange)

**AutoNektome** — пользовательский скрипт для [nekto.me/audiochat](https://nekto.me/audiochat), который автоматизирует поиск собеседников, добавляет голосовое управление, автогромкость, изменение голоса, проверку микрофона, анти-рекламу и кастомизацию интерфейса в одном удобном меню.

## 🆕 Что нового в 5.0.0

- 🧩 **Обновлённая панель:** Material-стиль, статус разговора, мини-виджет, сворачивание в шестерёнку, перетаскивание и изменение размера окна.
- 🎨 **Больше тем:** добавлены Original, GitHub Dark High Contrast, Catppuccin Mocha, Ayu Dark, Gotham, Rose Pine Moon, Gruvbox Dark и обновлённые Dracula, One Dark, Monokai, Nord.
- 🎙️ **Настройка голосовых команд из меню:** фразы теперь можно менять без редактирования кода — через кнопку **«Настроить фразы»**.
- 🎚️ **Проверка микрофона:** встроенный эквалайзер, запись до 8 секунд и прослушивание результата.
- 🛡️ **Блокировка рекламы и метрик:** скрытие рекламных блоков, мягкие заглушки счётчиков и безопасный режим, который не ломает загрузку сайта.
- 🧪 **Диагностика:** опциональные логи и `AutoNektomeDebug.dump()` для поиска проблем.

## ✨ Возможности

- 🔄 **Авторежим:** автоматический переход к новому собеседнику между завершёнными диалогами со звуковыми эффектами начала и конца беседы.
- 📊 **Счётчик разговоров:** показывает количество разговоров, текущую длительность и общее время общения. При наведении — статистика по длительности: 5, 15, 30 минут, 1, 2, 3, 5 часов.
- 🟢 **Статус общения:** в панели видно, что сейчас происходит: готово, поиск или активный разговор.
- 🎙️ **Голосовое управление:** управление чатом голосом. Список команд отображается в меню и настраивается в отдельном окне.

```javascript
const DEFAULT_VOICE_COMMANDS = {
    skip: ['скип', 'skip', 'скиф', 'скипнуть', 'кефир', 'далее', 'next'],
    stop: ['завершить', 'остановить', 'закончить', 'кумыс', 'стоп', 'stop'],
    start: ['чат', 'старт', 'поиск', 'начать', 'start']
};
```

Хотите изменить команды? Нажмите **«Настроить фразы»** в меню или см. раздел "Инструкции". ✂️

- 🔊 **Настройки звука:**
  - 🎤 **Самопрослушивание:** слушайте себя с регулировкой громкости (0.1–3.0).
  - 🎚️ **Автогромкость микрофона:** автоматическая настройка уровня микрофона.
  - 📢 **Автогромкость собеседника:** плавно выравнивает громкость собеседника.
  - 🎵 **Низкий голос:** изменение голоса через Web Audio / AudioWorklet с настройкой уровня (0–0.40).
  - 🔇 **Шумоподавление:** уменьшает фоновый шум.
  - 🔁 **Эхоподавление:** уменьшает эхо.
  - 🔈 **Управление звуком:** быстрые кнопки отключения микрофона и звука.
  - 🎛️ **Проверка микрофона:** эквалайзер, тестовая запись и прослушивание прямо в меню.
- 🌌 **Интерфейс:** компактная панель с переключателями, ползунками, статусом, мини-режимом и темами сайта.
- 🛡️ **Анти-реклама:** скрытие рекламных блоков, отключение метрик и безопасные заглушки счётчиков.
- 💾 **Сохранение настроек:** параметры, статистика, команды, тема, размер и позиция панели сохраняются в `localStorage`.

**Важно:** часть аудио-настроек применяется при новом доступе к микрофону, новом поиске или после перезагрузки страницы. Для надёжного применения низкого голоса, шумоподавления, эхоподавления и автоусиления микрофона перезагрузите страницу (F5 или Ctrl+R). 🔄

## 🛠 Установка

1. Установите менеджер скриптов:
   - [Tampermonkey](https://www.tampermonkey.net/) для Chrome, Firefox, Edge и др.
2. Включите "режим разработчика" в настройках расширений (`chrome://extensions/`).
3. В некоторых браузерах может понадобиться включить дополнительное разрешение на выполнение пользовательских скриптов. Для этого щёлкните правой кнопкой мыши по значку Tampermonkey -> выберите "Управление расширением" -> найдите пункт "Разрешить пользовательские скрипты". Включите его и перезапустите браузер.
4. Установите скрипт: [ссылка](https://update.greasyfork.org/scripts/498724/AutoNektome.user.js).
5. Перейдите на [nekto.me/audiochat](https://nekto.me/audiochat) — меню появится автоматически. ✅

## 📖 Как использовать

1. Зайдите на [nekto.me/audiochat](https://nekto.me/audiochat).
2. В правом верхнем углу появится меню **AutoNektome**:
   - Включите **Автопоиск**, чтобы скрипт сам переходил к следующему собеседнику после завершения диалога.
   - Используйте быстрые кнопки **🎤** и **🎧**, чтобы выключать микрофон или звук.
   - Активируйте **Голосовое управление** и используйте команды:
     - "Скип" / "Далее" / "Next" — пропустить собеседника.
     - "Завершить" / "Стоп" / "Stop" — выключить авторежим и остановить диалог.
     - "Чат" / "Старт" / "Поиск" / "Start" — включить авторежим и начать поиск.
   - Нажмите **«Настроить фразы»**, чтобы заменить голосовые команды на свои.
   - Откройте **Проверку микрофона**, чтобы посмотреть уровень сигнала, записать короткий тест и прослушать себя.
   - Настройте звук: самопрослушивание, автогромкость, шумоподавление, эхоподавление и низкий голос.
   - Выберите тему сайта: Original, GitHub Dark, GitHub Dark High Contrast, Catppuccin Mocha, Ayu Dark, Gotham, Rose Pine Moon, Gruvbox Dark, Dracula, One Dark, Monokai, Nord.
3. Панель можно перетаскивать, менять размер, сворачивать в шестерёнку или переключать в мини-виджет. ⚡

## 📝 Инструкции

1. **Изменение звуков уведомлений:**
   Откройте скрипт и в начале файла найдите блок "Настройка звуков уведомлений":

```javascript
const START_CONVERSATION_SOUND_URL = 'https://zvukogram.com/mp3/22/skype-sound-message-received-message-received.mp3';
const END_CONVERSATION_SOUND_URL = 'https://www.myinstants.com/media/sounds/teleport1_Cw1ot9l.mp3';
const START_SOUND_VOLUME = 0.4;
const END_SOUND_VOLUME = 0.3;
```

   Замените ссылки на свои `.mp3` или установите громкость `0`, чтобы отключить звук. Сохраните изменения.

2. **Смена голосовых команд через меню:**
   Включите **Голосовое управление** и нажмите **«Настроить фразы»**. Впишите слова через запятую или с новой строки. Сработает любое совпадение из списка.

3. **Смена голосовых команд вручную в коде:**
   Откройте скрипт и найдите блок `DEFAULT_VOICE_COMMANDS`:

```javascript
const DEFAULT_VOICE_COMMANDS = {
    skip: ['скип', 'skip', 'скиф', 'скипнуть', 'кефир', 'далее', 'next'],
    stop: ['завершить', 'остановить', 'закончить', 'кумыс', 'стоп', 'stop'],
    start: ['чат', 'старт', 'поиск', 'начать', 'start']
};
```

   Измените слова в массивах, добавьте или уберите команды. После сохранения обновите страницу.

4. **Темы сайта:**
   Темы берутся из папки `/themes` в репозитории. Если вы собираете проект вручную, убедитесь, что CSS-файлы тем лежат в этой папке и доступны по ссылкам из `THEME_REPO_BASE`.

5. **Диагностика проблем:**
   Откройте **Дополнительные настройки** -> включите **Диагностические логи**. После этого в консоли можно вызвать:

```javascript
AutoNektomeDebug.dump()
```

   Это покажет текущие настройки, счётчики событий и последние диагностические записи.

## 📋 Требования

- Современный браузер (Chrome, Yandex Browser, Edge, Vivaldi и другие Chromium-браузеры).
- Менеджер пользовательских скриптов, например Tampermonkey.
- Доступ к микрофону на сайте.
- Для голосового управления нужен браузер с поддержкой `webkitSpeechRecognition`. В Firefox и Arc голосовое управление может работать нестабильно или не работать.
- Если в Vivaldi голосовое управление выдаёт ошибку `network`, попробуйте отключить встроенный блокировщик рекламы и трекеров для `nekto.me`.
